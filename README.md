# scfetch
the ultimate successor to neofetch and betterfetch
## installation
for latest release:
```bash
pip install scfetch
```
for latest commit:
```bash
pip install git+https://github.com/sctech-tr/scfetch.git
```
## usage
```bash
scfetch
```
## what does it show
- os
- host
- kernel
- uptime
- arch
- shell
- resolution
- cpu
- gpu
- ram
- de/wm
- battery
- disk usage
- python version
- locale
- private ip
- public ip
- mac address
- colored bar to identify terminal colors
